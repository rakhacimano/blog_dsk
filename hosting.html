Cara Menghosting Websitemu ke UPT Jarkom PENS

Sebagai kampus teknologi, PENS memiliki banyak fasilitas untuk menunjang kegiatan pembelajaran, khususnya dalam bidang teknologi. Salah satunya adalah fasilitas hosting website atau sekaligus penyimpanan awan yang bernama Web FTP PENS.

Apasih Web FTP PENS itu? Singkatnya adalah layanan yang diberikan PENS untuk civitas akademika PENS guna menunjang kegiatan pembelajaran di bidang IT. Mahasiswa bisa mengunggah file maupun folder projek ke dalam layanan ini. Hosting lah ya istilah kerennya.

Web FTP PENS ini dapat digunakan selama mejadi mahasiswa aktif PENS dan memiliki kapasistas sejumlah 100GB

Hosting Website: Lalu gimana sih cara menggunakannya?

Yang kamu perlu siapkan cuman kuota internet atau jaringan internet aja kok. Caranya mudah banget, ikuti langkah-langkah di bawah ini ya!

  • Buat website statis menggunakan HTML CSS dan atau Javascript
  • Kunjungi tautan https://student.pens.ac.id/
  • Pilih menu Student Site
  • Setelah itu akan diarahkan ke situs EEPIS Student Portal
  • Klik pada tombol klik di sini
    • Jika sudah punya akun, langsung login aja pakai akun EEPIS
    • Jika belum, silahkan daftarkan akun EEPIS ke Web Portal
  • Jika login berhasil, upload projectmu dengan klik tombol upload folder
  • Tunggu hingga proses selesai, dan boom, hosting berhasil!
